2. **[HIGH]** Deploy skill needs CLI/TUI project-type detection — auto-detect from Cargo.toml binary targets + CLAUDE.md keywords. Prevents AskUserQuestion stalls (Pattern 3).

3. **[MEDIUM]** Add pre-flight validation to `solo-dev.sh` — check API key, rate limit status before entering main loop (Pattern 2).
